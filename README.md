<h1 align="center">GitWyvern: Scout Python Repos with AI</h1>
<p align="center">GitWyvern is an AI powered repository intelligence system for python based repositories. It's a beautifully orchestrated application that leverages local LLMs to summarize readme files, analyse python scripts, perform hybrid security scans (SAST+LLM) and create a repo awared chat system by leveraging RAG pipeline.</p>
