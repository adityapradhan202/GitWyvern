Here's the list of CLI commands you can use:

1. `python gitwyvern.py analyze [github_url]` - Analyzes code files.   
2. `python gitwyvern.py security [github_url]` - Performs a hybrid security scan (SAST + LLM).  
3. `python gitwyvern.py chat-wyvern [github_url]` - Creates and starts repo-aware chat system in the CLI itself.  
4. `python gitwyvern.py clear-cli-logs` - To clear the CLI logs.  
5. `python gitwyvern.py clear-app-logs` - To clear the application logs.
